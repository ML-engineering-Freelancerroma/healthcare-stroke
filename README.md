Этот репозиторий содержит в себе небольшое исследование влияния SMOTENC и RandomUnderSampler из библиотеки imblearn на данные с большим дисбалансом таргетных классов в бинарной классификации. Данные были взяты из общедоступного датасета (https://www.kaggle.com/datasets/aouatifcherdid/healthcare-dataset-stroke-data), разделены на валидационную и тестовую выборки в соотношении 75 на 25 с сохранением процента дисбаланса таргета (файл healthcare-stroke-prepare-for-ML.ipynb). Обработка признаков осуществлялась различными способами (такие как one-hot encoding, label encoding и ручное кодирование) (файл feat_prepare.ipynb). Стоит отметить, что для таких типов задач наиболее важны такие метрики как Recall, ROC-AUC, PR-AUC, поскольку важно определить как можно больше положительных случаев (в данном случае — пациентов с риском инсульта) и при этом не вызвать большое количество ложных срабатываний. Был проведен перебор оригинальных гиперпараметров для следующих моделей: KNeighborsClassifier, DecisionTreeClassifier, RandomForestClassifier из библиотеки scikit-learn, а так же для CatBoostClassifier из библиотеки catboost и XGBClassifier из библиотеки xgboost, MLPClassifier из билиотеки sklearn.neural_network при помощи GridSearch (файл train_orig.ipynb). Следующие этапы включали в себя обработку данных при помощи SMOTENC и RandomUnderSampler для минимизации дисбаланса и получения наилучшего результата (файлы train_SMOTENC.ipynb и train_RUS.ipynb соответственно). При обучении всех моделей использовалась стратификация StratifiedKFold. Более тонкая настройка гиперпараметров может выдать приемлемый результат, но потребует больших мощностей и больше времени для вычислений. По результатам метрик был проведен анализ и был сделан вывод о том, что RandomUnderSampler больше подходит для решения подобных задач при самых распростарненных настройках гиперпараметров моделей. В итоге была выбрана модель, показавшая лучший результат - RandomForestClassifier с RandomUnderSampler. Проведена повторная тренировка наиболее подходящих моделей (RandomForestClassifier, XGBClassifier, CatBoostClassifier) с расширенной сеткой RandomizedSearchCV из sklearn.model_selection и выбрана самая лучшая модель по всем показателям - CatBoostClassifier.
